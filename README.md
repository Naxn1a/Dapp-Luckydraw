# Dapp LuckyDraw

Try your luck by paying sepoliaETH and go get the prize!

## Installation

1. Clone the repo.
2. cd src/app/contract/
3. Deploy smartcontract and config address, abi.
    ```json
    {
        "address": "",
        "abi": ""
    }
    ```
4. Install NPM packages.
    ```sh
    npm install
    or
    bun install
    ```
5. Config dotenv.
    ```
    OWNER=""
    SECRET_KEY=""
    PRICE=""
    ```
6. Start.
    ```sh
    npm run dev
    or
    bun run dev
    ```
